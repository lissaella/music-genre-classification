Music Genre Classification
This repository features a Jupyter notebook dedicated to identifying music genres through machine learning. 
Key Highlights:
Exploratory Data Analysis: Visualizations to uncover patterns and insights within the dataset.
Feature Selection: Since the data contains 60 features, I used Logistic Regression to identify and rank the most impactful variables.
Predictive Modeling: A Random Forest classifier trained on the optimized feature set to predict genres accurately
